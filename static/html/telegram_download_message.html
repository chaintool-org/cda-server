You can download data for the last n days, n weeks, and n months using nd, nw, and nm respectively. Today (UTC) is included. You can also download data for a specific date range(DD-MM-YYYY). Both start date and end date are included. for example: /download 1d, /download 2d /download 1w, /download 2w /download 1m, /download 2m /download 26-12-2023 03-01-2024
