You can download data for the last *n* days, weeks, or months using `nd`, `nw`, and `nm`, respectively, with today (UTC) included. Additionally, you can download data for a specific date range (DD-MM-YYYY), where both the start and end dates are included. For example: /download 1d, /download 2d /download 1w, /download 2w /download 1m, /download 2m /download 26-12-2023 03-01-2024
